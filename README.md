# Data Analysis with Python

Welcome to my repository dedicated to tracking my learning journey through freeCodeCamp's Data Analysis with Python curriculum. 

My goal is to master data manipulation, statistical analysis, and visualization tools to build a strong foundation for transitioning into **Machine Learning and AI Engineering**.

---

## 📂 Repository Structure

The project is organized by core technical topics using a clean, flat folder system:

* [foundations_of_data_analysis](./foundations_of_data_analysis) — Core differences between Data Analysis and Data Science, and understanding open vs. closed frameworks.
* [python_fundamentals](./python_fundamentals) — Working with Pandas Dataframes, checking dataset dimensions (`.shape`), and identifying data distribution shapes or outliers.
